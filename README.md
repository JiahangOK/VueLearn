## 知识点  
- MVVM  
model+view model+view，双向绑定  
- 模板
- 插值操作
    - Mustache  
    {{}}
    - v-once  
    静态内容，不会随数据改变
    - v-html  
    展示html元素， 字符串里面有标签
    - v-text  
    {{}}等效
    - v-pre  
    设置不解析，比如让{{}}原封不动的显示出来
    - v-cloak  
    解析前有这个属性，解析后没有这个属性，经常和css配合使用
